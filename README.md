Cursor Website – Landing Page Redesign

This project is a frontend recreation of the Cursor AI landing page, built using pure HTML and CSS. The objective of this project is to practice modern frontend development and understand how professional landing pages are structured and styled.

🚀 Features

Clean and modern landing page design

Responsive layout structure

Custom fonts and visual assets

Organized and maintainable folder structure

Built entirely with HTML and CSS (no frameworks)

🛠 Technologies Used

HTML5 – Structure and layout

CSS3 – Styling and responsiveness

SVG & PNG Assets – Icons and graphics

├── assets/

│
└── README.md

🎯 Project Goal

The purpose of this project is to:

Improve frontend development skills

Practice recreating real-world landing pages

Understand layout techniques used in professional websites

Work with custom fonts and visual assets
